# Physics Project – GlowScript

## Overview
This project is a physics assignment created using **GlowScript**.  
It demonstrates key physics concepts such as center of mass.

## Project Details
- **Language:** GlowScript (VPython)  
- **Purpose:** To simulate and visualize physics phenomena in a simple, interactive way  
- **Key Features:**  
  - Motion simulation  
  - Velocity and acceleration calculations  
  - Clear inline comments explaining the code  

## How to Use
1. Open the project code (`project.py`) in a GlowScript environment: [GlowScript Link](https://www.glowscript.org/#/user/situnafsileshi/folder/MyPrograms/program/Physicsproject)  
2. Run the program to see the physics simulation in action  
3. Explore and modify parameters to understand the effects on motion
